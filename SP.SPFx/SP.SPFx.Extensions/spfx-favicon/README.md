# spfx-favicon

## Summary

This SharePoint Framework extension allows you use a custom favicon.ico for a site.

The code for this solution was isnspired by the solution from [https://github.com/joaoferreira/SPFx-favicon](https://github.com/joaoferreira/SPFx-favicon).

## Minimal Path to Awesome

- Clone this repository
- Ensure that you are at the solution folder
- in the command-line run:
  - **npm install**
  - **gulp serve**

## Features

To use a Favicon for your site you can add this solution as an app to any site. The solution will attempt to use a file called **favicon.ico** located in the **Site Assets** library of the site.
